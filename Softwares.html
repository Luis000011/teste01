Softwares

Photoshop

É um editor de fotos, imagens, vídeos e mídias. As vantagens deste software de edição apresentam efeitos e resultados únicos, além da organização que ele oferece, importando facilmente fotos e vídeos, assim como arquivos de mídia de um meio móvel. Outra característica essencial é a correção de imagens com a rapidez de alguns cliques.

Requisitos mínimos:

Processador: Intel ou AMD com suporte a 64 bits; processador de 2 GHz ou mais rápido.

Sistema operacional: Microsoft Windows 7 com Service Pack 1 (64 bits).

Memória RAM: 2 GB ou mais de RAM.

Disco Rígido: 6 GB.

Winrar

Winrar é um software compactador e descompactador de dados. Possibilita comprimir e descomprimir arquivos nos formatos RAR, ZIP e outros formatos de compactação. Ele oferece as melhores vantagens quando se trata de compactação. As funções do Winrar resultam em economia de espaço em disco e de custos de transmissão de dados e de tempo de trabalho, por agilizar a troca de arquivos.

Requisitos mínimos:

Processador: Processador de 2 GHz ou superior.

Sistema Operacional: Windows 7 ou superior.

Memória RAM:64 MB.

Disco Rígido: 32 MB.

7-Zip

7-Zip é um compactador de arquivos de código aberto para o sistema operacional Microsoft Windows e Linux. O programa compete diretamente com os programas de código-fechado WinZip e WinRAR e está disponível para download grátis. 7-Zip é muito mais leve que a maioria dos programas do gênero. Sua eficiência também é superior a boa parte dos programas, capaz de comprimir mais os documentos e economizar ainda mais espaço.

Requisitos mínimos:

Processador de 2GHz no mínimo.

Memória RAM: 256 MB.

Disco Rígido: 50 MB.

Sistema Operacional: Windows Vista ou superior.

Discord

Discord é um aplicativo gratuito de comunicação que permite que você converse por voz, vídeo e texto com amigos, comunidades de jogos e desenvolvedores. Ele possui centenas de milhares de usuários, o que faz com que ele seja uma das formas mais populares de se conectar com as pessoas online.

Requisitos do Sistema:

Sistema Operativo: Windows 8, 10, ou 11.

Processador: 2GHz ou mais.

Memória: 2 GB de RAM.

Placa gráfica: 64MB.

Espaço no disco: Requer 200 MB de espaço livre.

Linkedin

O LinkedIn serve para criar conexões, relacionamentos que possibilitem a visualização da sua marca e sua relevância/autoridade na sua área de atuação, servindo, portanto, tanto para empresas, como para profissionais. Ou seja, é uma rede para relacionamentos profissionais que pode resultar em anúncio de vagas, conquista de emprego, networking, estabelecimento de parcerias e alcance de clientes.

Requisitos mínimos: Praticamente inexistentes! É um programa leve e simples de usar, exige pouco da máquina e é garantido que seu computador aguenta.

Skype

O Skype é o software que permite conversar com o mundo todo. Milhões de pessoas e empresas usam o Skype para fazer chamadas de vídeo e voz gratuitas entre dois usuários, bem como chamadas em grupo, enviar mensagens de chat e compartilhar arquivos com outras pessoas no Skype.

Requisitos mínimos:

Sistema operacional: Windows 7 ou superior.

Processador: Dual-core 1,5 GHz ou superior.

Memória RAM: 2 GB ou mais.

Placa Gráfica: DirectX v9.0 ou superior.

Sistemas Operacionais

Windows 10

O Windows 10 é o sistema operacional padrão da maioria dos computadores atuais, embora já tenham sido lançadas versões mais atualizadas. O sistema operacional é o que dá a vida ao PC, basicamente, sendo que o computador se torna inútil para qualquer outra função sem ter um instalado. O Windows 10 é referência nesse quesito é altamente recomendado.

Requisitos mínimos:

Processador: Processador de 1 GHz ou mais rápido ou Sistema em um chip (SoC).

Memória RAM: 1 gigabyte (GB) para 32 bits ou 2 GB para 64 bits.

Disco rígido: 16 GB para um sistema operacional de 32 bits ou 32 GB para um de 64 bits.

Placa gráfica: DirectX 9 ou posterior com driver WDDM 1.0

Windows 10 Pro

A versão Pro — de profissional — é a mais completa de todas e abrange mais recursos para desenvolvedores, por exemplo. Já a versão Pro é voltada para usuários domésticos, que não vão precisar de recursos em rede, remotos ou mais complexos.

Requisitos mínimos:

Processador: de 1 GHz.

Memória RAM: 1GB para 32 bits.

Disco Rígido: 16 GB para 32 bits.

Placa gráfica: DirectX 9.

Linux

Diferente do Windows o Linux já vem com um conjunto básico de compiladores, como por exemplo o GCC. Linux é uma ótima operação para quem quer programar, você poderá programar em PHP, JAVA, RUBY, PYTHON E MUITAS OUTRAS LINGUAGENS. Algumas distribuições Linux já disponibilizam estes ambientes por padrão.

Requisitos mínimos: Processadores: com 1 GHz.

Memória RAM: 768 MB.

Disco Rígido: 8GB.

Android

Voltado agora para os smartphones, o Android é o sistema operacional móvel mais utilizado do mundo, e, em 2013, possuía a maior porcentagem das vendas mundiais desse tipo de sistema. É referência global e ainda conta com integração em programação, sendo inclusive baseado no núcleo Linux. Os requisitos mínimos desse caso são simplesmente inexistentes. Basta você verificar se o celular que você está comprando vem com Android e voilá.

MacOs, IOS

Desenvolvido pela Apple, o MacOS é o sistema operacional específico para os computadores da Apple, os Macs. É o segundo sistema operacional mais famoso do mundo, perdendo apenas para o Windows. Ele conta com uma interface bem simples, sendo uma ótima escolha pra novatos no mundo da tecnologia, mas sua principal vantagem é rodar muito bem todos softwares gráficos.

Requisitos mínimos:

Sistema Operacional: OS X v10.6 ou superior.

Memória RAM: 2 GB de RAM.

Disco rígido: 8 GB de armazenamento livre.

É necessário acesso à internet e Apple ID para certos recursos também.

Pacote office

One Drive

É um aplicativo de armazenamento na nuvem, logo, seus principais recursos incluem armazenamento em nuvem, capacidade de compartilhar arquivos e pastas com colegas de trabalho, controle de documentos por administradores e sincronização de conteúdo de uma biblioteca para a plataforma.

Requisitos mínimos:

Sistema operacional: Versão de 32 ou 64 bits do Windows 10/11, Windows 8/8.1 ou Windows 7.

Processador: 1,6 GHz ou superior.

Memória: 1 GB de RAM ou superior.

Resolução: mínimo de 1024 x 576.

Conexão com a Internet: Sim.

Word

É um software que une vantagens de um processador de textos com os recursos oferecidos pela interface gráfica do Windows. O Word dispõe das seguintes funções: Copia e move fragmento de texto, parágrafos e desenhos. lnserção simplificada de gráficos, planilhas e desenhos, variedades de tipos e tamanhos de fontes, incluindo símbolos gráficos, etc.

Requisitos mínimos:

Processador: Processador de 2 núcleos com 1,6 GHz.

Sistema operacional: Windows 11 ou Windows 10 para PC; Mac OS para Mac.

Memória RAM: 4 GB (64 bits), 2 GB (32 bits) de RAM para PC; 4 GB de RAM para Mac.

Espaço no disco rígido: 4 GB de espaço disponível em disco para PC, 10 GB para Mac.

Monitor: Resolução de 1024 x 768 para PC, 1280 x 800 para Mac.

Placa de vídeo: DirectX 10 para a aceleração gráfica por hardware para PC.

Exel

O Microsoft Excel é um editor de planilhas produzido pela Microsoft para computadores que utilizam o sistema operacional Microsoft Windows, dentre outros. Dispositivos relacionados. Seus recursos incluem uma interface intuitiva e capacitadas ferramentas de cálculo e de construção de tabelas que, juntamente com marketing agressivo, tornaram o Excel um dos mais populares aplicativos de computador até hoje.

Requisitos mínimos: Sistema operacional: Windows 10.

Processador: de 1 GHz ou mais veloz.

Memória RAM: 1 GB (32 bits) ou 2 GB (64 bits) de RAM.

Disco Rígido: 16 GB de espaço livre em disco.

Placa Gráfica: Microsoft DirectX 9 com driver WDDM.

Power Point

O Power Point é o programa do Pacote Office que permite criar apresentações de slides. Contando com animações e transições, o Power Point é uma excelente escolha para apresentação de projetos, plano de fundo para palestras ou até mesmo anúncios em empresas.

Requisitos mínimos:

Processador: Intel Celeron E1200

Memória RAM: 4 GB (64 bits), 2 GB (32 bits) de RAM para PC; 4 GB de RAM para Mac.

Sistema operacional: Windows 11 ou Windows 10 para PC; Mac OS para Mac.

Disco Rígido: 4 GB de espaço disponível em disco para PC, 10 GB para Mac.

Placa gráfica: Placa de vídeo DirectX 10 para a aceleração gráfica por hardware para PC.

Navegadores

Google Chrome

O Google Chrome é um navegador de internet desenvolvido pela Google. Foi lançado pela primeira vez em setembro de 2008, para o Microsoft Windows, e mais tarde foi portado para Linux, Mac, iOS e Android. É o navegador mais utilizado na atualidade e seu sistema de buscas é impressionante de tão preciso.

Requisitos mínimos:

Processador: Intel Pentium 4 ou superior.

Sistema operacional: Windows Xp ou superior.

Memória RAM: 512 MB.

Disco Rígido: 350 MB.

Mozila Firefox

Outro navegador muito recomendado também é o Mozila Firefox, diferentemente do Chrome, o layout do Firefox permite a personalização a partir de uma grande biblioteca de extensões e um grande foco na segurança de seus usuários. É uma boa opção de navegador também.

Requisitos mínimos: Sistema operacional: versão 7, 8 ou 10 (32 ou 64 bits).

Processador: Pentium 4 ou mais recente.

Memória RAM: 512 MB (32 bits) ou 2 GB de RAM (64 bits).

Disco Rígido: 200 MB de espaço no disco rígido.

Opera Gx

É um navegador criado pra games que possui o GX Control, um painel de controle que ajuda a acompanhar e controlar a performance. Ativando-o, usuários conseguem limitar o uso de componentes do computador, como memória RAM, processador e consumo de rede pelo navegador.

Requisitos mínimos:

Sistema Operacional: Windows 7.

Processador: Intel Pentium 4 ou AMD Athlon 64.

Memória RAM: 2GB.

Disco Rígido:33,23 MB (Windows) 53,3 MB (Mac OS X) 5.0,2 MB (Linux x86-64).

Acesso à internet: Sim.
